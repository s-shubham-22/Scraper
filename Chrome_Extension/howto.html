1. Select Option

Select the any one option from dropdown by which you want to scrap the Website.

Example: Using Selector or Using XPath

2. Select Number of Elements

Enter Number of Elements you want to Scrap.

For example, if you want to Scrap Name and Scraping (thus 2 Elements) so you will enter 2 in Input Box

3. Submit

Click on Submit Butto and you'll get more input fields according to your initial input.

4. Fill the Input Fields and Submit

Fill Up URL textbox and other Label and Selector Text Box.

You Have to Input the URL of the Website which you want to Scrap.

In Label textbox(es) you have to enter the label name for your scraped data column (e.g. Name, Rating, Price etc.)

In Selector textbox(es) you have to enter the CSS Selector or XPath of the data you want to Scrap. (Which you can get from Inspect tab in Chrome Developer Tools)

After Filling out the Fields, hit the Submit Button.

5. Hit the Download Button

After Submittig, it will take some time to scrap data.

After some time Download Button will Appear.

By clicking on that button you can Download the Scraped Data CSV File

6. Your File will get Downloaded

After clicking the download button your file will get downloaded.

IF YOU DON'T KNOW HOW TO GET THE CSS SELECTOR OR XPATH

1. Go to Inspect

Right CLick on Page and Select bootom most Option "Inspect"

2. Find Element and Copy Selector

Find the element in HTML content and copy it's CSS Selector / XPath by right clicking on it's HTML Content
